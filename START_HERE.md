## â“ Common Questions

### Q: How do I change my name?
**A:** Edit `src/pages/Home.jsx` line 25

### Q: How do I change colors?
**A:** Edit `tailwind.config.js` lines 8-10

### Q: How do I add a new page?
**A:** 
1. Create `src/pages/NewPage.jsx`
2. Add route in `src/App.jsx`
3. Add link in `src/components/Navbar.jsx`
See [ADVANCED_CUSTOMIZATION.md](./ADVANCED_CUSTOMIZATION.md) for details

### Q: How do I deploy?
**A:** Run `vercel` (see [INSTALLATION_GUIDE.md](./INSTALLATION_GUIDE.md) Step 7)

### Q: Does it work on mobile?
**A:** Yes! Fully responsive. Test with: `npm run dev` then visit on phone

### Q: Can I change the theme?
**A:** Yes! Navbar has theme switcher. Edit themes in `src/store/themeStore.js`

### Q: What if port 3000 is busy?
**A:** Run `npm run dev -- --port 3001`
